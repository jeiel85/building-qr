# 08. Store and Monetization

## 상용화 방향

초기에는 유료 결제보다 무료 배포 + 프리미엄 기능 확장 가능성을 열어두는 것이 좋습니다.

## 수익화 후보

### 1. 무료 + Pro Export

무료:
- 기본 프리셋
- 1024px export
- 기본 PNG 저장

Pro:
- 2048/4096px export
- 투명 배경
- 상업용 템플릿
- 워터마크 제거
- 프리셋 확장

### 2. 일회성 유료 앱

장점:
- 단순함
- 개인정보/서버 부담 적음

단점:
- 무료 도구와 경쟁 어려움
- 구매 전 체험 장벽

### 3. 웹 무료 + 앱 유료

웹에서 체험 후 앱은 고해상도 저장/프리셋 기능으로 유료화할 수 있습니다.

## 초기 추천

초기 상용 배포 버전에서는 다음을 추천합니다.

```txt
무료 출시
→ 사용성 검증
→ 고해상도 export / 프리셋 팩 / 워터마크 제거를 유료화 후보로 설계
```

## 스토어 등록 정보 초안

### 앱 이름 후보

- Blossom QR
- Sakura QR Art
- Bloom QR Maker

### 짧은 설명

링크를 감성적인 벚꽃 QR 아트로 만들어보세요.

### 긴 설명 초안

Blossom QR은 평범한 QR 코드를 감성적인 3D 벚꽃 블록 아트로 바꿔주는 QR 생성 앱입니다.

블로그, 프로필, 초대장, 행사 안내, 명함, 앱 다운로드 링크 등을 더 아름답게 공유할 수 있습니다.

주요 기능:
- URL/텍스트 QR 생성
- 3D 벚꽃 QR 아트 보기
- 스캔 가능한 2D QR 보기
- PNG 이미지 저장
- 모바일 공유
- 다양한 색상/스타일 프리셋
- 입력값을 서버로 전송하지 않는 로컬 생성 방식

주의:
일부 긴 텍스트는 QR 구조가 복잡해져 스캔이 느릴 수 있습니다. 안정적인 스캔을 위해 짧은 URL 사용을 권장합니다.

## 개인정보 방침 방향

- 계정 없음
- 서버 저장 없음
- 입력한 URL/텍스트 외부 전송 없음
- 광고 SDK 없음
- 분석 SDK 없음
- 로컬 저장만 사용
