# 09. License and Risk

## 원본 데모 관련 원칙

원본 GitHub 데모는 아이디어 참고용입니다. 이 프로젝트에서는 원본 코드를 복사하지 않습니다.

## 안전한 재구현 원칙

- 코드 복사 금지
- 파일 구조 그대로 복제 금지
- shader 코드 복사 금지
- 색상/상수/수식 그대로 복제 금지
- UX 아이디어는 참고하되 독자 구현
- README 또는 About 화면에 inspiration credit을 넣을 수 있음
- 상업 배포 전 원본 라이선스 전문 재검토

## 라이선스 리스크

원본 라이선스는 상업 프로젝트 내 사용은 허용하지만, 코드 재배포/재판매/경쟁 애니메이션 라이브러리 제작/컴포넌트 단독 배포 등을 제한합니다.

이 프로젝트가 “QR 아트 생성 앱” 자체를 제품 가치로 삼는 경우, 원본 코드를 직접 사용하는 것은 리스크가 있습니다. 따라서 반드시 재구현해야 합니다.
